The Beautiful Game

I am a pretty big soccer fan and you may catch me wearing jerseys from any number of leagues across the world.

To be a big fan you have to stay up on the latest news so it probably comes as no surprise then that I have a preferred app for doing just that:

OneFootball

I cannot regurgitate the insane amount of articles that they put out, otherwise I'd start up a competing service. But the enthusiast in me would like to at least provide some content here for like-minded individuals.

Enjoy!!

Note: I've consumed a free API that someone was kind enough to set up on RapidAPI.com for sharing game highlights and laid them out evenly using Angular's FlexLayout module here. The embed divs and iframes were already returned by the api so I had to use Angular's [innerHtml] property and bypass the DomSanitizer to allow the content to be used.
